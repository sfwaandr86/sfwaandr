Skill Saya

HTML

HTML, singkatan dari "HyperText Markup Language," adalah bahasa markup yang digunakan untuk membuat halaman web. HTML adalah salah satu elemen utama dalam pengembangan web, dan itu adalah bahasa yang digunakan untuk mengatur struktur dan konten dasar halaman web

CSS

CSS, atau "Cascading Style Sheets," adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak elemen-elemen dalam halaman web. Dengan CSS, Anda dapat mengontrol warna, font, spasi, dan banyak aspek tampilan lainnya pada halaman web

JAVASCRIPT

JavaScript adalah bahasa pemrograman yang digunakan untuk mengembangkan aplikasi web yang interaktif dan dinamis. JavaScript berjalan di sisi klien (di peramban web pengguna) dan memungkinkan Anda mengontrol perilaku elemen HTML, berinteraksi dengan pengguna, dan melakukan berbagai tugas lainnya
